# Кликабельное подтверждение команд (модальный диалог с кнопками)

**Контекст.** Сейчас подтверждение — текст в нижней панели, ответ только с
клавиатуры. Делаем аккуратный центрированный модал с кнопками.

**Шаги:**

1. В `ui/input.rs` (или новый `ui/confirm.rs`): в режиме `Confirming` рисовать
   поверх чата центрированный блок (ширина `min(70, area.width - 8)`, высота по
   контенту, `Clear` под ним):
   - заголовок: `⚠ Confirm command` (danger, bold) — или ` Confirm command `
     без символа, ASCII-безопасно;
   - объяснение (`fg`), при `destructive` — строка-предупреждение (danger);
   - сама команда: строка с префиксом `$ `, цвет `warning`, при длинной команде —
     wrap;
   - пустая строка;
   - кнопки: `[ Approve (a) ]` и `[ Deny (d) ]` — прямоугольники с фоном
     `theme.surface`, текст success/danger соответственно; между ними 3 пробела;
   - выбранная по умолчанию кнопка (Deny — безопасный дефолт) подсвечена
     инверсией; Tab/←/→ переключают выбор, Enter активирует выбранную.
2. Рамка модала — единственное место, где рамка уместна: `Borders::ALL`,
   скруглённые углы `BorderType::Rounded`, цвет `danger` при destructive,
   иначе `warning`.
3. Прямоугольники кнопок записывать в `app.confirm_button_areas: Vec<(Rect, bool)>`
   при рендере (bool = approve). В `handle_mouse`: `Down(Left)` по кнопке →
   `respond_to_confirmation(approve)`. Hover (`MouseEventKind::Moved`) —
   подсветка кнопки под курсором (хранить `hovered_button: Option<bool>`).
4. Клавиатурное поведение прежнее: `a/y/e/д-эквиваленты` — approve,
   `d/n/…` — deny, Ctrl+C — deny + quit. **Изменение:** голый Enter теперь
   активирует выделенную кнопку (по умолчанию Deny), а не approve — это
   безопаснее; отразить в help-баре.
5. Нижняя панель ввода в режиме Confirming показывает приглушённую строку
   `waiting for confirmation…` — layout не прыгает.

**DoD:**
- Подтверждение — модал по центру, кликабельные кнопки Approve/Deny работают.
- Tab/стрелки переключают кнопку, Enter активирует, буквенные хоткеи работают.
- Деструктивная команда визуально отличается (красная рамка + предупреждение).
- Layout остальных зон при появлении модала не смещается.
