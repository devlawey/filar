# Выделение текста мышью и копирование в буфер

**Контекст.** Главная боль mouse-capture в TUI: пропадает нативное выделение
терминала. Возвращаем его своими руками — иначе пользователь не сможет
скопировать вывод команды.

**Шаги:**

1. Добавить зависимость `arboard = "3"` в `crates/tui/Cargo.toml`
   (кроссплатформенный clipboard; Windows поддерживается из коробки).
2. В `App`: `pub selection: Option<Selection>`, где
   `struct Selection { anchor: (usize /*line*/, usize /*col*/), head: (usize, usize) }` —
   координаты в пространстве строк `layout_cache` (не экрана!), чтобы выделение
   переживало скролл.
3. `handle_mouse`:
   - `Down(Left)` в области чата (не по toggle/скроллбару) → `selection = Some`
     с anchor=head в позиции клика, `mouse_drag = Some(Selection)`;
   - `Drag(Left)` → обновлять `head`; если курсор у верхней/нижней кромки
     области чата — автоскролл на 1 строку за тик;
   - `Up(Left)`: если anchor == head → снять выделение (обычный клик).
4. Рендер: в `chat.rs` после получения видимых строк накладывать
   `theme.selection_bg` на диапазон выделения (пересечение стилей span-ов:
   проще всего пере-стилизовать посимвольно только затронутые строки).
5. Копирование: `Ctrl+Shift+C` или автоматически по `Up` (выбрать: авто-копия
   по отпусканию — поведение как в терминалах с «copy on select»; сделать
   авто-копию, это элегантнее). Извлечь текст выделенных строк из
   `layout_cache` (сырое содержимое `Line` → `to_string`), для средних строк —
   целиком, для крайних — по колонкам. После копирования — короткий тост:
   строка `· copied` в статус-баре на ~1.5 сек (хранить `toast: Option<(String, Instant)>`).
6. Двойной клик — выделить слово, тройной — строку (отслеживать интервал
   < 400 мс между Down по той же позиции).
7. Любое новое сообщение/перестройка кэша со смещением индексов → снять
   выделение (просто и предсказуемо).

**DoD:**
- Drag мышью выделяет текст в чате, отпускание копирует в системный буфер,
  появляется тост `copied`.
- Двойной клик выделяет слово, тройной — строку.
- Выделение переживает скролл колесом; сбрасывается при новых сообщениях.
- Сборка на Windows проходит, clipboard работает (ручная проверка).
